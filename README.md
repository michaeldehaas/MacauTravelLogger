# Macau Travel Logger

An Android app that automatically detects and logs 
work trips to Macau using geofencing.

## Features
- Auto border detection via Android Geofencing API
- Manual check-in / check-out override
- Rolling 6-month compliance counter (45-day limit)
- Monthly reports exported as PDF and CSV
- Android share sheet for sending reports to HR

## Tech Stack
Kotlin · Jetpack Compose · MVVM · Room · Hilt · 
Geofencing API · iTextG · OpenCSV
